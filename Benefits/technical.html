Technical Benefits


Light

Free software and GNU/Linux distros tend to be lighter than ther proprietary counterparts, allowing them to concentrate all the power of the machine on the most important part, processing


Security

Statistically, the amount of malware and spyware developed for linux is lower than any other, and the few that have been known have been counteracted quickly. Also, since you have complete control over the system and its applications, it is relatively easy to solve it as a user if you have knowledge on the subject.


The world runs on linux

Most servers, supercomputers and labs run Linux as their operating system. This wouldn't be an argument as such (here we don't approve fallacies ad verecundia) if it were not for the protocols, commands and packages that make communication between Linux so easy and fast.


Bash

Although someone say the opposite and it's extremly rare for some users of proprietary software, the terminal is one of the most useful and important tools that a computer can offer. Allowing you to handle it perfectly and much more efficiently than with graphical methods.
